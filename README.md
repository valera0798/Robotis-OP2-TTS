# Robotis OP2 Text-to-Speech (TTS)

Text-to-Speech application for Robotis OP2 robot.

## Robotis OP2 Configuration

- Lubuntu 12.04 LTS
- Python 2.7.3
- festival 2.1 (festvox-ru package installed)
- mpg123 0.2.13
